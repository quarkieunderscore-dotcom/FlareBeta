Connect to Firebase

FlashChat needs a free Firebase Realtime Database to sync messages between users. Takes about 3 minutes to set up — done once, forever.

1
Go to console.firebase.google.com and sign in. Click Add project, name it anything (e.g. "flashchat"), click through and create it.
2
In the left sidebar, click Realtime Database → Create database → pick any region → select Start in test mode → Enable.
3
Click the ⚙️ gear icon → Project settings → scroll to Your apps → click </> (Web) → register with any nickname → copy the firebaseConfig block and paste it below.
{ apiKey: "AIzaSy...", authDomain: "your-app.firebaseapp.com", databaseURL: "https://your-app-default-rtdb.firebaseio.com", projectId: "your-app", ... }
